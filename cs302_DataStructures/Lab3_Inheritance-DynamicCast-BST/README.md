# Program 3 : Game Add / Search 

This program manages data in a binary search tree, allowing users to store many different types of data and have access to function such as insert, removal and display based on their types using dynamic binding.

## Installation

Upload the cpp files h files and makefile into a linux server, once uploaded enter the word make to compile using the makefile.

## Features

Dynamic Casting : Using dynamic cast this program is able to detect the Node type and make the function behave depending on their type.

Dynamic Binding: using dynamic binding, the binding is at run time and will call the correct functions based on their type.

Binary Search Tree: Data is stored in a binary search tree, all 3 derived classes are stored in this structure.

## Usage

The program starts with 0 games entered, the user must enter games to be able to search using the find game function. Once the find game function is called,
The program will travel to the node and dynamic binding to all the correct display functions, and work on the correct type nodes.
